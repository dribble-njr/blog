HTML5 总共以下新特性，但不支持 IE8 及以下版本浏览器。

1. 语义化标签
2. 增强型表单
3. 视频和音频
4. Canvas 绘图
5. SVG 绘图
6. 地理定位 API
7. 拖拽 API
8. WebWorker
9. WebStorage
10. WebSocket
11. 选择器 `query.selector` 和 `query.selectorAll`
